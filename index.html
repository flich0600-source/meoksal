Q
temp메일로 qwen 계정을 어떻게 빠르게 대량생성하나요?
6회
답변 보기
▼
조교답변

크롬 게스트 프로필로 temp-mail 접속 후 가입하고, 게스트 프로필을 열고 닫으면 매번 새 메일주소가 발급됩니다. delete로 재생성하되 25개 이상이면 제한이 걸리니 다시 게스트모드로 반복하면 됩니다. 조호메일보다 열 배 빠릅니다.

Q
qwen(콴)이 무엇이고 왜 쓰나요?
4회
답변 보기
▼
경험자답변

중국 기반 무료 img-to-video AI로, 계정당 하루 5초 영상 5개 생성이 가능합니다. 그록 사용량 감소·불안정 이슈의 대체 수단이며, 계정 30개 확보가 숙제로 예상됩니다.

Q
썸네일 강의는 어디에서 볼 수 있나요?
3회
답변 보기
▼
조교답변

멱살강의 2강에 나오며, 노샘 피드백 강의에서도 계속 다룹니다. 멱살강의는 카페에 올라오지 않고 1·2일차는 다운로드, 3일차는 다시보기 링크로 시청합니다.

Q
veo 모델은 어떤 걸 골라야 하나요?
2회
답변 보기
▼
조교답변

720P 6초 기준 generate는 품질 최상·비용 초당 0.4달러, fast는 품질 중상·초당 0.1달러, lite는 품질 하·초당 0.05달러입니다. 사용자가 몰리면 lite가 오히려 더 빠를 수 있습니다.

Q
플로우플로우 작업을 다른 PC에서 이어서 할 수 있나요?
2회
답변 보기
▼
조교답변

Q
페이블(Fable)은 계속 무료로 쓸 수 있나요?
2회
답변 보기
▼

Q

Q

Q
강의는 언제·어디서 진행하고 녹화본은 제공되나요?
18회
답변 보기
▼
수강환경방장, 조교

강의는 매일 진행되며 시간은 보통 저녁 7시 이후(초기 첫 줌은 8시 지연 시작)에 그날 공지됩니다. 초반에는 줌으로 진행했으나 화면 전환이 편한 유튜브 라이브 스트리밍으로 전환됐습니다. 종료 후 1시간 내 녹화본이 업로드되며, 지난 줌 녹화본은 구글 드라이브 폴더에서 다운받아 볼 수 있습니다.

🔧 해결방안
  1. 1. 저녁 7시쯤 올라오는 강의 시간 공지를 확인한다
  2. 2. 유튜브 라이브 링크로 접속해 시청한다
  3. 3. 놓쳤다면 종료 1시간 후 업로드되는 녹화본 또는 구글 드라이브 폴더에서 다시 본다
Q
flow(플로우)에서 이미지를 대량 생성하면 '비정상적인 활동'으로 감지돼 막힙니다. 어떻게 푸나요?
9회
답변 보기
▼
기술문제방장, 조교

이미지를 30~110개 이상 생성하면 IP·계정 기록과 연동돼 차단됩니다. 오류 재생성을 반복하면 더 심해집니다. IP 교체가 핵심 대응이며, 고객센터에 온라인셀러·마케터로 아이템스카우트 조회 때문에 IP가 밴됐다는 사유로 교체를 요청하거나, 크롬 프로필을 바꿔 다른 구글 계정으로 시도합니다. '자동 저장'에서 추적된다는 의견도 있어 자동 다운로드 해제와 오토세이브 off도 시도해볼 수 있습니다. 그래도 안 풀리면 3~5일 쉬는 게 안전합니다.

🔧 해결방안
  1. 1. 고객센터에 IP 밴 사유(마케터·셀러 조회)로 IP 교체를 요청한다
  2. 2. 크롬 프로필을 새로 만들고 다른 구글 계정으로 로그인해 확장 재설치 후 시도한다
  3. 3. 유동IP면 공유기·랜선을 뽑고 재접속한다
  4. 4. 주소창 왼쪽 정보(i) 버튼에서 자동 다운로드 해제 + 오토세이브 off를 시도한다
  5. 5. 계속 막히면 3~5일 쉬었다 재시도한다
Q
챌린지를 이미 시작했는데 다시 시작해도 되나요? 인증 유예는 언제까지인가요?
9회
답변 보기
▼
기타방장, 조교

네. 실패·누락 카운트 없이 모두 5일(일요일)부터 동일하게 재시작할 수 있으며 페르소나 셋팅부터 다시 해도 됩니다. 잘 이어가고 계신 분은 계속 진행해도 됩니다. 인증 유예기간은 7월 5일까지이며, 5일부터 업로드를 시작하면 그때부터는 유예 없이 바로 적용됩니다. 개인 사정으로 못 할 경우 버터떡님께 1:1톡으로 유예 신청이 가능합니다.

🔧 해결방안
  1. 1. 이미 시작했어도 5일부터 재시작할지, 계속 이어갈지 선택한다
  2. 2. 5일 전까지는 인증 유예가 적용되니 강의 듣고 재정비한다
  3. 3. 개인 사정이 있으면 버터떡님께 1:1톡으로 유예를 신청한다
Q
1:1 오픈프로필을 어떻게 카카오 기본프로필로 변경(닫기)하나요?
8회
답변 보기
▼
수강환경조교

PC 기준 참가자 리스트에서 내 프로필을 클릭한 뒤 프로필 관리에서 변경합니다. 오픈프로필만 닫혀 있으면 되며, 1:1 채팅을 닫고 카카오 기본프로필로 바꾸는 것이 운영 규칙입니다.

🔧 해결방안
  1. 1. PC에서 참가자 리스트를 연다
  2. 2. 내 프로필 클릭 → 프로필 관리로 이동한다
  3. 3. 오픈프로필 1:1 채팅을 닫고 카카오 기본프로필로 변경한다
Q
클로드는 어떤 요금제·모델로 쓰고, 결제는 미리 해야 하나요?
7회
답변 보기
▼
비용/결제조교, 경험자

대본 작업은 오푸스(높음) 기본 세팅이면 맥스 요금제에서 막히지 않습니다. 클로드는 맥스 5배(약 110달러)를 추천하며, 그록은 선택입니다. 신규계정 3일 무료기간을 써보고 결정해도 됩니다. 맥스에서 프로로 낮추면 대본 하나에 주간 한도 14%가 소모될 만큼 빡세지만 짧은 영상은 1일 1영상이 가능하며, 새로 나온 소넷이 대안이 될 수 있습니다. 울트라 코드는 헤비한 코딩용입니다.

🔧 해결방안
  1. 1. 우선 클로드 맥스 5배(약 110달러) 요금제를 준비한다
  2. 2. 대본은 오푸스(높음) 모델로 세팅한다
  3. 3. 무료기간을 활용해 소넷 등 대안 모델을 테스트해본다
  4. 4. 프로 요금제는 한도가 빡세니 짧은 영상 위주로 운용한다
Q
qwen(콴)이 무엇이고 생성 한도는 어떻게 되나요?
6회
답변 보기
▼
도구사용방장, 조교, 경험자

qwen(콴)은 중국 기반 무료 img-to-video AI(chat.qwen.ai)로, 그록의 사용량 감소·불안정 이슈를 대체하는 수단입니다. 계정당 하루 5초 영상 5개 생성이 가능하며 8시간 주기로 초기화됩니다. 한 프로필에서 계정만 갈아끼우면 막힐 수 있어, 크롬 프로필을 분리해 프로필별로 다른 콴 계정을 돌리면 하루 수십~100개까지 생성할 수 있습니다. 프로필은 구글 로그인 없이 콴 계정만으로도 사용 가능합니다.

🔧 해결방안
  1. 1. chat.qwen.ai에 구글 계정으로 로그인한다
  2. 2. 계정당 5개·8시간 초기화 한도를 염두에 둔다
  3. 3. 크롬 프로필을 여러 개 분리하고 프로필마다 다른 콴 계정을 물린다
  4. 4. 프로필을 번갈아 돌려 하루 대량 생성한다
Q
temp메일로 qwen 계정을 어떻게 빠르게 대량생성하나요?
6회
답변 보기
▼
도구사용조교, 경험자

크롬 게스트 프로필로 temp-mail에 접속해 가입하고, 게스트 프로필을 열고 닫으면 매번 새 메일주소가 발급됩니다. delete로 재생성하되 25개 이상이면 제한이 걸리니 게스트모드를 다시 열어 반복하면 계속 생성할 수 있습니다. 조호(zoho)메일은 번호 하나당 6개 생성이 가능하지만, temp메일 방식이 열 배 빠릅니다.

🔧 해결방안
  1. 1. 크롬 게스트 프로필로 temp-mail.org에 접속한다
  2. 2. 발급된 임시메일로 qwen에 가입한다
  3. 3. 게스트 프로필을 열고 닫아 새 주소를 받아 반복한다
  4. 4. 25개 제한이 걸리면 게스트모드를 다시 열어 이어간다
Q
이미지가 다 생성됐는데 flow 패널에 아무것도 안 뜨거나 업로드가 안 됩니다.

답변 보기

기술문제조교

이지워커·그록그록·플로우플로우 등 저장경로를 지정하는 확장이 동시에 활성화되면 경로 충돌이 납니다. 사용하는 확장 하나만 남기고 나머지는 비활성화(토글 off)해야 합니다.

🔧 해결방안
  1. 1. 크롬 확장 관리로 이동한다
  2. 2. 경로지정 확장(이지워커·그록그록·플로우플로우 등) 중 사용할 하나만 남긴다
  3. 3. 나머지 확장은 토글을 꺼서 비활성화한다
  4. 4. flow를 새로고침해 재시도한다
Q
페이블(Fable)은 계속 무료로 쓸 수 있고 토큰은 왜 이렇게 빨리 닳나요?
5회
답변 보기
▼
비용/결제조교, 경험자

페이블은 7월 7일까지만 무료 개방되고 이후에는 API로 별도 과금됩니다(추후 서버가 충분해지면 구독제 전환 예정). 페이블 전용 게이지가 별도로 생겨 사용 시 'Fable 게이지'와 '모든 모델 게이지'가 동시에 차오르기 때문에 전체보다 빨리 소진되며, 조금만 써도 5시간 세션 제한이 걸릴 수 있습니다. 소모용으로 쓰긴 아까우니 대본은 소넷 등 저렴한 모델로 뽑고 워싱·리팩토링·대본스킬 제작 등에만 쓰는 절약법이 권장됩니다.

🔧 해결방안
  1. 1. 7월 7일 이전까지 무료 개방 기간을 활용한다
  2. 2. 1차 대본은 소넷 등 저렴한 모델로 뽑는다
  3. 3. 페르소나 워싱·리팩토링·검증 단계에만 페이블을 사용한다
