Q
커스텀 스타일 화풍이 칙칙하고 추상적으로만 나올 때 개선 방법은?

답변 보기
▼
조교답변

튜브워커 스타일추출기는 저성능 모델 기준이라 한계가 있으니, 클로드에 레퍼런스 이미지와 커스텀스타일 등록 화면을 함께 주고 500자 이내 화풍 프롬프트를 요청하세요. 이때 장면묘사는 넣지 말고 화풍만 담아야 하며, 파파고로 번역해 장면묘사 혼입 여부를 검수하고 플로우에서 바로 테스트하며 다듬습니다.

Q

답변 보기
▼

3강은 강의 파일이 아니라 유튜브 링크이며, 상단고정 공지에 링크가 걸려 있습니다.

Q
플로우에서 무료로 이미지를 대량 생성하는 안전한 방법은?
2회
답변 보기
▼
방장답변

이미지는 플로우로 뽑습니다. 계정마다 한도가 있으니 1장씩 대기속도 6초 이상으로 안전하게 걸어두고, 100장 단위로 씬을 나눠 계정을 돌려 진행하세요. 100장 생성 후 막히면 프롬프트 파일에서 앞 100장을 지우고 101번 씬부터 다른 계정으로 이어받으면 됩니다.

Q
두 구글 계정을 프로필 분리해 동시에 돌려 이미지를 뽑아도 되나요?
1회
답변 보기
▼
조교답변

안 됩니다. 두 계정을 프로필 분리해 동시에 돌리면 비정상 감지에 바로 걸립니다. 씬을 나눠(1~100 1계정, 101~200 2계정) 순차로 진행해야 하며, 안전 마지노선은 100~150장입니다.

Q
FGV로 영상화(그록)까지 가능한가요?
2회
답변 보기
▼
조교답변

Q
플로우 나노바나나프로는 생성량이 금방 차나요?

Q

1회

Q
경제 카테고리에 추천하는 TTS 목소리는?
1회
답변 보기
▼
방장답변

Q

qwen(콴)이 무엇이고 왜 쓰나요?
4회
답변 보기

경험자답변

중국 기반 무료 img-to-video AI로, 계정당 하루 5초 영상 5개 생성이 가능합니다. 그록 사용량 감소·불안정 이슈의 대체 수단이며, 계정 30개 확보가 숙제로 예상됩니다.

3회

veo 모델은 어떤 걸 골라야 하나요?
2회
답변 보기

조교답변

720P 6초 기준 generate는 품질 최상·비용 초당 0.4달러, fast는 품질 중상·초당 0.1달러, lite는 품질 하·초당 0.05달러입니다. 사용자가 몰리면 lite가 오히려 더 빠를 수 있습니다.

Q
강의(줌·유튜브 라이브)는 언제, 어디서 진행하고 녹화본·다시보기는 제공되나요?
19회
답변 보기
▼
수강환경조교, 방장

초기에는 줌으로 진행하다 화면 전환이 편한 유튜브 라이브로 전환되었습니다. 시작 시각은 보통 저녁 7~8시경으로 유동적이며, 강의 종료 후 1시간 내에 녹화본이 업로드됩니다. 방장이 페이블 스킬 제작에 집중한 기간에는 줌을 중단했다가 8일부터 재개하기로 했고, Q&A·라이브도 8일부터 진행합니다. 지난 줌·라이브 녹화본은 구글드라이브 폴더와 상단고정 유튜브 링크에서 확인할 수 있습니다.

🔧 해결방안
  1. 1. 저녁 7~8시경 시작하는 라이브를 기다린다(당일 시각은 공지에서 확인).
  2. 2. 놓쳤다면 종료 1시간 후 업로드되는 녹화본을 구글드라이브 폴더에서 시청한다.
  3. 3. 3강 등 일부 강의는 상단고정 공지의 유튜브 링크로 다시보기한다.
  4. 4. Q&A·정규 라이브는 8일부터 재개되므로 그 전까지 강의를 모두 보고 질문을 메모해둔다.
Q
플로우/구글에서 '비정상적인 활동(이상한 행동) 감지'로 이미지·영상 생성이 막힐 때 어떻게 대응하나요?
13회
답변 보기
▼
기술문제방장, 조교, 경험자

대량 생성·다계정 동시 실행·의심 IP가 원인입니다. 한 번 걸린 계정은 최소 3일 이상 쉬어야 하고, 반복되면 구글이 IP까지 기억하므로 여러 구글 계정을 여유롭게 돌려 써야 합니다. 두 계정을 프로필 분리해 동시에 돌리면 즉시 감지되니 씬을 나눠 순차 진행하고, 안전 마지노선은 100~150장입니다. IP는 통신사 고객센터에 온라인셀러·마케터로 조회가 많아 막혔다며 교체를 요청하거나, 랜선 재연결·라우터 위치 변경·IP타임 랜카드·모바일 핫스팟·외부 와이파이로 바꿉니다. VPN은 구글이 바로 차단하므로 부적합하고, 단말기 고유값(IMEI)이 같으면 우회 효과가 제한되어 USB 5G 라우터 같은 별도 회선이 확실합니다.

🔧 해결방안
  1. 1. 걸린 계정은 즉시 재시도하지 말고 최소 3일 묵힌다.
  2. 2. 구글 계정을 여러 개 확보해 100~150장 단위로 씬을 나눠 순차 사용한다(동시 실행 금지).
  3. 3. 자동 다운로드·오토세이브를 끄고 1장씩 6초 이상 대기로 생성한다.
  4. 4. IP는 고객센터 교체 요청, 랜선·라우터·IP타임 랜카드·핫스팟·외부 와이파이로 변경한다(VPN 금지).
  5. 5. 반복되면 3~5일 쉬거나 별도 회선(USB 5G 라우터)을 사용한다.

  2. 2. 이미 잘 진행 중이면 그대로 이어간다.
  3. 3. 개인 사정이 생기면 방장에게 1:1톡으로 유예를 신청한다.
  4. 4. 4일 연속 미업로드 시 처음부터 재시작되므로 매일 1영상을 유지한다.
Q
qwen(콴)은 무엇이고, 계정을 어떻게 빠르게 대량 생성하나요?
11회
답변 보기
▼
도구사용조교, 방장, 경험자

qwen(콴)은 중국 기반 무료 img-to-video AI(chat.qwen.ai)로, 계정당 하루 5초 영상 5개를 만들 수 있어 불안정한 그록의 대체 수단입니다. 무료 한도를 늘리려면 계정 30개 확보가 숙제로 제시되었습니다. temp-mail.org 임시 이메일을 크롬 게스트 프로필로 열고 닫으면 매번 새 주소가 발급돼 조호메일보다 열 배 빠르게 가입할 수 있고, 30개를 1시간 이내에 만들 수 있습니다. delete로 재생성하다 25개 이상이면 제한이 걸리니 게스트모드를 다시 열어 반복합니다.

🔧 해결방안
  1. 1. chat.qwen.ai에 구글 계정으로 접속해 사용한다.
  2. 2. 크롬 게스트 프로필로 temp-mail.org에 접속해 임시 이메일을 받는다.
  3. 3. 게스트 프로필을 열고 닫으며 새 주소로 qwen에 반복 가입한다.
  4. 4. 25개 이상에서 제한이 걸리면 게스트모드를 다시 열어 이어간다(목표 30개, 1시간 내 가능).
