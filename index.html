Q
스타일 프롬프트가 500자를 넘거나 계속 오류가 나요.
1회
답변 보기
▼
조교답변

스타일 프롬프트는 500자 이내(쉼표 포함), 특수문자 제외여야 합니다. 그리고 스타일 프롬프트에는 '장면 묘사'가 들어가면 안 되고 화풍만 담아야 합니다. 만들어진 프롬프트를 파파고에 넣어 번역해 장면묘사가 섞였는지 검수하세요.

Q
플로우(오토워커) 이미지 생성을 돈 안 내고 안전하게 뽑는 방법은?

🔧 해결방안
  1. 1. 크롬 게스트 프로필로 temp-mail에 접속해 새 임시 메일주소를 발급
  2. 2. 그 주소로 qwen(chat.qwen.ai)에 가입
  3. 3. 게스트 프로필을 닫았다 다시 열면 새 메일이 발급되므로 반복해 계정을 늘림
  4. 4. delete로 재생성하되 25개 이상에서 제한이 걸리면 게스트모드를 다시 열어 반복

🔧 해결방안

  2. 2. 계정마다 한도가 있으니 프로필을 분리해 여러 계정을 돌림빵
  3. 3. 200장 중 100장에서 막히면 프롬프트 파일 앞 100장을 지우고 101번 씬부터 다른 계정으로 이어 생성

Q
대본 작성 시 클로드는 어떤 모델과 effort로 설정하고, 요금제별 한도는 어떤가요?

도구사용경험자, 조교

대본은 오푸스(effort 하이)가 기본 세팅이며 맥스 요금제면 막히지 않고 뽑을 수 있습니다. 대본 작성 자체가 아주 복잡한 작업은 아니라 하이 정도면 무난합니다(최대치로 쓴다는 의견도 있음). 다만 맥스에서 프로로 낮추면 대본 하나에 주간 한도의 14%가 소모될 만큼 빡세며, 짧은 영상은 그래도 1일 1영상이 가능합니다. 울트라 코드는 헤비한 프로그램 코딩용이고, 새로 나온 소넷이 대안이 될 수 있어 테스트가 필요합니다.

🔧 해결방안
  1. 1. 대본용 모델은 오푸스로 설정하고 effort는 하이로 지정
  2. 2. 가능하면 맥스 요금제를 사용해 한도 여유 확보
  3. 3. 프로 요금제는 대본 1개에 주간 한도 소모가 크므로 짧은 영상 위주로 운용하거나 소넷 등 대안 모델을 테스트
Q
FGV는 무엇이고 어떻게 쓰며, 이미지·영상 모델은 무엇을 골라야 하나요?
7회
답변 보기
▼
도구사용조교, 방장

FGV는 FlowGrokVeo, 즉 플로우·그록·베오를 합친 확장프로그램(아아X후파 합본)으로 이미지뿐 아니라 영상까지 생성합니다. 'f'가 플로우플로우이므로 플로우플로우를 따로 먼저 쓸 필요 없이 FGV만 쓰면 되고, 기존 플로우플로우는 삭제해도 됩니다. 후파님 이지워크와 결과물은 동일하며 그록 UI 변경분을 개선한 버전입니다. 영상은 베오가 좋지만 정책이 빡빡해 안 되는 건 그록으로 시도하고, 이미지는 imagen4는 비추천, 바나나2가 무난하고 그림체 일관성도 좋으며 글씨가 필요할 때만 pro를 씁니다.

🔧 해결방안
  1. 1. 플로우플로우를 따로 쓰지 말고 FGV 하나만 설치·사용(기존 플로우플로우는 삭제 가능)
  2. 2. 이미지 모델은 바나나2를 기본으로, 글씨가 필요한 경우에만 pro 사용, imagen4는 지양
  3. 3. 영상은 베오를 우선 쓰되 정책 차단되면 그록으로 대체
  4. 4. 그록은 유료로 써야 하며, 그록 UI 변경 시 FGV를 재설치·zip 재공유로 대응

콘텐츠제작조교, 방장

그림이 칙칙하고 어둡게만 나온다면 커스텀 스타일에 하드한 명암 등 어둡게 만드는 요소가 들어간 것이므로 스타일 프롬프트를 확인·수정해야 합니다. 튜브워커 스타일 추출기는 저성능 수준까지만 뽑아 한계가 있으니, 클로드에 레퍼런스 이미지와 커스텀스타일 등록 화면 캡쳐를 주고 '이 화풍으로 500자 이내 이미지 생성 프롬프트를 작성해줘'라고 요청한 뒤 플로우에서 테스트하며 다듬는 편이 정확합니다. 스타일 프롬프트는 500자 이내(쉼표 포함)·특수문자 제외이며, 장면 묘사는 빼고 화풍만 담아야 합니다.

🔧 해결방안
  1. 1. 클로드에 레퍼런스 이미지 + 커스텀스타일 등록화면 캡쳐를 주고 500자 이내 화풍 프롬프트 요청
  2. 2. 완성 프롬프트를 파파고로 번역해 장면 묘사가 섞였는지 검수(화풍만 남기기)
  3. 3. 500자 이내·특수문자 제외 규칙을 지켜 등록 후 플로우에서 테스트하며 수정
  4. 4. 화풍을 바꿀 때는 스토리보드를 다시 만들 필요 없이, 프롬프트 내보내기 텍스트 파일을 클로드에 주고 기존 스타일 프롬프트를 새 것으로 치환
Q
이미지 등장인물의 얼굴·옷 일관성을 유지하려면(마스코트 설정 포함) 어떻게 하나요?
6회

▼
콘텐츠제작조교, 방장

인물 통일성이 계속 바뀌면 스타일 프롬프트에 국가·인종을 넣거나 마스코트를 별도로 만들어 쓰면 거의 비슷하게 나옵니다. 다만 커스텀 스타일에 '인물은 이렇게 표현하라'는 캐릭터 묘사를 길게 넣는 것은 추천하지 않는데, 프롬프트가 길어지면 AI가 혼동해 오히려 결과가 나빠질 수 있기 때문입니다. 마스코트는 채널 목록에서 해당 채널에 마우스를 올려 뜨는 '채널 수정'에서 커스텀 스타일 생성·마스코트 이미지 업로드로 설정합니다(새로 만들기가 아니라 채널 수정으로 접근).
